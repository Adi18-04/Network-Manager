# Network-Manager
Simulated Cyber Forensics Investigation using Kali Linux (WSL)

This experiment simulates a cyber forensics scenario where network traffic is monitored and analyzed. The Python-based implementation captures packets, processes intercepted signals, and produces reports that reflect observed activity.

**Setup**

- Environment: Kali Linux running on Windows Subsystem for Linux

- Python Virtual Environment (venv) used for dependency management

- Custom Python scripts used for packet capture and analysis

**Execution**

- The project is executed inside a Python virtual environment

- The script captures network packets from the system interface

- Intercepted traffic is processed and stored

**Output**

- Captured packet data

- Generated reports based on analyzed network activity
